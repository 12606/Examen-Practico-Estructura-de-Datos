# Examen-Practico-Estructura-de-Datos
Desarrollar una aplicación web con Spring Boot que exponga una API REST para ejecutar, sobre grafos definidos por el usuario, los algoritmos de fuerza bruta y voraces estudiados: DFS, BFS, profundidad limitada, profundidad iterativa, A*, Kruskal y Prim. El sistema debe devolver los resultados en formato JSON y ofrecer una interfaz para probarlos
